Unit->반환문이 없는 함수   

unit은 데이터형식이 아닌 특수한 상황을 표현하려는 목적.   

함수에서 반환문이 없음을 명시적으로 나타낼 때 Unit타입을 사용   

val data1:Unit=Unit
fun some(): Unit{   
    println(10+20)   
}   

함수를 선언할때 반환타입을 생략하면 자동으로 Unit이 적용.

    println(10+20)
