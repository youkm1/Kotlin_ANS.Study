  제트팩은 API 레벨의 호환성 문제를 해결해준다. 
  ### androidx.appcompat
  이 라이브러리를 쓰면 자동적으로 호환 문제를 해결하기 때문에 다음과 같은 코드를 쓰지 않아도 된다.
  ```
  if (Build.VERSION.SDK_INT >= ....) { ... } 
  ```
